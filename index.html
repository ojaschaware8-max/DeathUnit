I developed a PC based team deathmatch shooter in Unreal Engine as a fun personal project to showcase my coding skills and logical thinking. Few months ago I thought i should try to build my own game. I thought it would be a easy task and it will improve my logical thinking. I thought if it is improving my skills then it's good enough reason to start. So i got to work. Game Features

Dynamic FPS System

Dynamic first-person shooter (FPS) system refers to a set of game mechanics, AI, collision of Environment with character and many more. These systems are designed to make combat feel more responsive,smooth.

Good Graphics

Good graphics consist of high-resolution, clear, and immersive visuals, realistic textures, and smooth lighting. It has adjustable Graphics to get good Performance.

AI

Scripted AI refers to non-player character (NPC) behaviors and game systems governed by pre-programmed, static rules. It has rule based behavior.
